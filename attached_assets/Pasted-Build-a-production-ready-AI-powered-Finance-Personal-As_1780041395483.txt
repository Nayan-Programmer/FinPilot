Build a production-ready AI-powered Finance Personal Assistant web application called "FinPilot".

Purpose:
The application is designed for finance professionals, CFOs, finance heads, accountants, and business owners. It should function as an intelligent finance copilot capable of analyzing financial data, generating reports, answering finance-related questions, and automating repetitive finance tasks.

Tech Stack:

* Frontend: React + TypeScript + Tailwind CSS
* Backend: FastAPI (Python)
* Database: PostgreSQL
* Authentication: JWT + Role-Based Access Control
* File Storage: Local storage with future cloud compatibility
* Charts: Chart.js
* AI Integration: OpenAI API
* Excel Processing: Pandas + OpenPyXL
* PDF Processing: PyPDF
* Deployment Ready
